# Jump King Frame Trainer (Unofficial)

This is an unofficial fan-made training tool for Jump King.  
It visualizes jump charge timing based on a fixed 60fps logic.

## Features
- Keyboard / Mouse / Gamepad supported
- Frame counting is fixed at 60fps
- Rendering uses requestAnimationFrame

## Notes
- Visual tick marks may appear misaligned depending on your display environment.
- The displayed frame numbers are always accurate.
- This tool does not modify the game.
